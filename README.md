# Operating System Security and Hardening

Academic cybersecurity project focused on operating system security concepts using Linux.

## Objectives
- Apply basic operating system security principles.
- Implement system hardening techniques.
- Manage users, permissions and access control.
- Analyze system logs for security-related events.

## Security Measures Implemented
- User and group management.
- Permission hardening (files and directories).
- SSH service hardening.
- Least privilege configuration using sudo.
- Review of active services.
- Basic system auditing using logs.

## Tools and Technologies
- Linux (Ubuntu / Kali)
- Bash scripting
- systemctl
- chmod, chown
- SSH
- Log analysis

## Cybersecurity Concepts
- Operating system security
- Hardening
- Access control
- Authentication
- Auditing
